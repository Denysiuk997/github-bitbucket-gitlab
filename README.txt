Система Time-Tracking. Администратор закрепляет за пользователем
Активность. У пользователя может быть одна или несколько Активностей.
Пользователь отмечает кол-во затраченного времени на каждую активность.
Пользователь может отправить запрос на добавление/удаление
Активности.


Для установки необходимо 
подключить сервер, например(Tomcat 9.0.20)
Подключить БД, настройка которой располагается в файле META-INF/context.xml ,например (mySQL)

Для запуска необходимо открыть страницу в браузере, в моем случае( http://localhost:8080/TimeTracking_war_exploded/)
Пройти регистрацию и авторизацию после чего использовать приложение

Сделать пользователя администратором возможно только при ридактировании поля role в тиблице customer (role ="admin")
